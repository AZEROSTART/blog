## 欢迎来到我的博客

这是我博客的第一篇文章，从今天起，我会在这里记录我的技术探索和生活感悟。

## 为什么要写博客？

写博客对我来说有几个重要意义：

1. **整理思路** — 写作是最好的思考方式，把脑子里模糊的想法变成清晰的文字。
2. **记录成长** — 技术日新月异，记录下来才能看见自己的成长轨迹。
3. **分享交流** — 也许你的一篇文章，正好帮助了某个在搜索答案的人。

## 这个博客是怎么搭建的？

这个博客使用 [Jekyll](https://jekyllrb.com/) 构建，部署在 [GitHub Pages](https://pages.github.com/)。整个过程非常简单：

- 写 Markdown 文章
- Push 到 GitHub 仓库
- GitHub Actions 自动构建并发布
